# MyLife — Личный менеджер

Приложение для управления личными финансами, задачами и ежедневными привычками.

## Функции

- 💰 **Финансы** — учёт доходов и расходов, график за 7 дней, разбивка по категориям  
- 📋 **Задачник** — задачи с приоритетами, дедлайнами и категориями  
- ✅ **День** — ежедневный чек-лист привычек с прогресс-баром

## Установка на iPhone

1. Открой ссылку на GitHub Pages в браузере **Safari**
2. Нажми кнопку «Поделиться» (квадрат со стрелкой вверх)
3. Выбери **«На экран "Домой"»**
4. Нажми **«Добавить»**

Готово! Приложение появится на рабочем столе как обычное приложение.

## Данные

Все данные хранятся локально на твоём устройстве. Ничего не отправляется в интернет.
